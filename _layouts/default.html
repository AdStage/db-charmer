DbCharmer

ActiveRecord Connection Magic Plugin

Rails plugin that significantly extends ActiveRecordâ€™s ability to work with multiple databases and/or database servers:

  • Simple ActiveRecord model connections management (switch_connection_to to switch to separate servers/databases)
  • Ability to easily control queries execution (Model.on_* methods to choose a connection for your queries)
  • Automated master/slave queries routing (SELECTs go to a slave, DML queries are handled by the master)
  • Multiple database migrations with very flexible query routing controls
  • Simple database sharding functionality with multiple sharding methods (value, range, mapping table)
